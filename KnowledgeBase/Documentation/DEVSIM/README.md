# DEVSIM documentation

This directory contains solver documentation, not research papers.

- Primary reference: https://devsim.net/
- Official documentation index: https://devsim.org/introduction.html#documentation
- Archived PDF: `devsim-manual-2.10.0.pdf`
- Documentation version: see `VERSION`

KORANI should use the HTML manual for section-level API retrieval and the PDF
as a local, versioned fallback. Research papers remain under
`ReferencePaper/TCAD/`. Solver examples should be stored separately under
`KnowledgeBase/Examples/DEVSIM/` if they are added later.
